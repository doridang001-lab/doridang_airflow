### 파일명 자동 생성 규칙

문서 주제에서 핵심 키워드를 추출해 한글 슬러그를 만든다.
- 공백 → `_`, 특수문자 제거, 최대 15자
- 예시: "손씻는방법", "배민_마케팅분석", "2026_1분기_영업보고"
- 최종 파일명: `{슬러그}_1.html`, `{슬러그}_2_애니메이션없는_ppt.pptx`

## 실행 순서

### 1단계: 문서 입력 처리

`$ARGUMENTS`가 파일 경로면 Read로 읽는다. 아니면 $ARGUMENTS 자체를 문서 내용으로 사용한다.
내용이 없으면 사용자에게 "발표 자료로 만들 문서를 붙여넣어 주세요." 라고 요청한다.

---

### 2단계: 문서 분석 → 슬라이드 설계

다음 항목을 먼저 파악한다:
- **핵심 메시지**: 이 발표가 전달하고자 하는 1문장 결론
- **발표 목적**: 보고 / 제안 / 공유 / 설득 중 분류
- **대상 청중**: 경영진 / 팀원 / 고객 / 전체 중 추정

슬라이드 구조는 두 가지 중 문서 성격에 맞는 것을 선택:
- **보고형**: 표지 → 문제 및 배경 → 목표 → 핵심 내용 → 기대효과 → 다음 단계
- **제안형**: 표지 → 선결론 → 문제 및 배경 → 목표 → 분석 결과 → 제안

슬라이드 설계 원칙:
- 한 슬라이드 = 하나의 핵심 메시지
- 원문 복붙 금지, 발표 흐름에 맞게 재구성
- 수치/날짜/사실관계는 원문 그대로 유지
- 표, 비교, KPI 박스, 흐름도를 적극 활용

사용자에게 다음을 먼저 출력:
1. 문서 핵심 요약 (3줄 이내)
2. 추천 슬라이드 목차 (번호 + 슬라이드 제목 + 한줄 설명)
3. 각 슬라이드 초안 (내용 요약)

### 3단계: 1.html 생성

아래 사양에 맞는 완전한 단일 HTML 파일을 생성한다.

**기술 요구사항:**
- 순수 HTML/CSS/JS — 외부 라이브러리 없이 단일 파일
- 16:9 비율 (1280×720 기준)
- 방향키(←→) 및 Space 키로 슬라이드 이동
- 진행바 (상단 또는 하단)
- 슬라이드 번호 표시 (현재/전체)
- URL hash 기반 이동 (#1, #2, ...)
- 슬라이드 전환: fade 또는 slide 애니메이션
- 슬라이드 내 요소 순차 등장 (클릭/키 누를 때마다 하나씩)

**폰트 크기 — 반드시 vw 단위 사용 (em/px 금지)**

슬라이드가 뷰포트에 맞게 스케일되므로 모든 텍스트는 `vw` 기준으로 작성해야 가독성이 확보된다.

| 요소 | vw 값 |
|------|-------|
| 표지 메인 타이틀 | 5.5vw ~ 6.5vw |
| 표지 서브타이틀 | 2.0vw |
| 슬라이드 제목 | 3.2vw |
| 섹션 태그 (STEP BY STEP 등) | 1.2vw |
| 본문 / 카드 제목 | 1.8vw |
| 본문 설명 / 소본문 | 1.4vw |
| KPI 수치 | 4.0vw |
| KPI 라벨 | 1.2vw |
| 하단 안내 / 캡션 | 1.2vw |
| 슬라이드 번호 카운터 | 1.1vw |

**디자인 기준:**
- 다크 배경(#0d0d0d 또는 #111) 기반 프레젠테이션 스타일
- 강조색: #00e5a0 (민트 그린) — 제목, 키워드, 포인트 수치
- 본문 텍스트: 흰색 계열
- 표지 슬라이드: 서브타이틀(소제목) + 큰 메인 타이틀 + 날짜/소속 정보
- 내용 슬라이드: 상단 슬라이드 제목 + 본문 영역
- KPI/수치 강조 박스, 비교 표, 단계 흐름도 등 시각 요소 포함

저장 경로: `C:\Users\민준\OneDrive - 주식회사 도리당\data\report\PowerPoint\{슬러그}_1.html`

### 4단계: 2_애니메이션없는_ppt.pptx 생성

아래 Python 스크립트를 작성하고 실행해 PPTX를 생성한다.
`py -3 <스크립트 경로>` 로 실행.

**PPTX 요구사항:**
- python-pptx 사용
- 슬라이드 크기: 33.87cm × 19.05cm (16:9 와이드스크린)
- HTML과 동일한 슬라이드 수/내용/구조
- 애니메이션 없음 (정적 문서)
- 보고서처럼 읽히는 레이아웃: 제목 + 내용이 명확히 구분
- 배경: 어두운 단색(#0d0d0d) 또는 흰색 중 문서 성격에 맞게
  - 경영진 보고 → 흰색 배경, 깔끔한 보고서 스타일
  - 발표/제안 → 다크 배경, HTML과 동일 스타일
- 표지: 타이틀 + 서브타이틀 + 날짜
- 내용 슬라이드: 제목 텍스트박스 + 본문 텍스트박스 + 필요시 표/도형
- 표가 있는 슬라이드는 python-pptx의 add_table로 추가
- 글꼴: 맑은 고딕 또는 Malgun Gothic

스크립트는 임시 경로에 저장 후 실행:
`C:\airflow\.claude\skills\ppt\temp_pptx_gen.py`

저장 경로: `C:\Users\민준\OneDrive - 주식회사 도리당\data\report\PowerPoint\{슬러그}_2_애니메이션없는_ppt.pptx`
