## Implementation Steps

### 1. 날짜별 다운로드 로직을 헬퍼로 추출
현재 `download_okpos_card_test`(약 847~913줄)의 "날짜 세팅 → 검색 → 매장행 대기 → 팝업 → 엑셀 다운로드" 블록(858~908줄)을 아래 헬퍼로 분리한다. 로그인/브라우저 기동은 세션당 1회만 유지(헬퍼 밖).

```python
def _download_one_date(driver, wait, download_dir: Path, sale_date: str) -> Path | None:
    """한 날짜의 승인현황 엑셀을 내려받아 경로 반환. 매장 행이 없으면(승인 0건) None."""
    driver.get(OKPOS_CARD_APPROVAL_URL)
    time.sleep(2)
    _dismiss_alert(driver)
    _set_date_hyphen(driver, wait, sale_date, "startDate")
    _dismiss_alert(driver)
    _set_date_hyphen(driver, wait, sale_date, "endDate")
    _dismiss_alert(driver)
    _click_search_btn(driver, wait)
    time.sleep(3)
    _dismiss_alert(driver)
    try:
        _wait_for_store_in_results(driver, wait, OKPOS_CARD_STORE["name"], OKPOS_CARD_STORE["shopCd"], download_dir)
    except TimeoutException:
        logger.warning("OKPOS card: 매장 행 없음(승인 0건 추정) sale_date=%s → skip", sale_date)
        return None
    _click_store_row_in_results(driver, wait, OKPOS_CARD_STORE["shopCd"], OKPOS_CARD_STORE["name"], download_dir)
    _wait_for_popup(driver, wait)
    existing_files = {p for p in download_dir.iterdir() if p.is_file()}
    _click_excel_button(driver, wait)
    downloaded = _wait_for_download(
        download_dir, existing_files, timeout=DOWNLOAD_TIMEOUT,
        expected_suffixes={".xlsx"}, filename_predicate=_filename_matches,
    )
    if downloaded is None:
        candidates = sorted(
            [p for p in download_dir.iterdir() if p.is_file() and _filename_matches(p)],
            key=lambda p: p.stat().st_mtime, reverse=True,
        )
        downloaded = candidates[0] if candidates else None
    return downloaded
```

### 2. 누락 lookback 날짜 계산 헬퍼
`_load_store_history` + `_canonicalize_card_df`로 승인 존재 날짜 집합을 구해, primary 기준 lookback 창에서 승인 0건인 날짜만 반환한다. `min_date`(=`OKPOS_CARD_MIN_VALIDATE_DATE`) 이전은 제외.

```python
def _missing_lookback_dates(primary: str, lookback_days: int = 7,
                            min_date: str = OKPOS_CARD_MIN_VALIDATE_DATE) -> list[str]:
    anchor = datetime.strptime(primary, "%Y-%m-%d").date() + timedelta(days=1)
    min_dt = datetime.strptime(min_date, "%Y-%m-%d").date()
    targets = [
        (anchor - timedelta(days=i)).strftime("%Y-%m-%d")
        for i in range(1, lookback_days + 1)
        if (anchor - timedelta(days=i)) >= min_dt
    ]
    history = _load_store_history()
    present: set[str] = set()
    if not history.empty:
        history = _canonicalize_card_df(history)
        if "승인구분" in history.columns and "sale_date" in history.columns:
            ap = history[history["승인구분"].astype(str).str.strip() == "승인"]
            present = set(ap["sale_date"].astype(str).str.strip())
    return [d for d in targets if d not in present and d != primary]
```

### 3. download 태스크에서 primary + 누락분 루프 수집·적재
`download_okpos_card_test`를 다음 흐름으로 재구성한다. 브라우저 1세션에서 primary를 먼저 받아 기존 XCom 계약(`downloaded_path`, `sale_date`)을 유지하고, 누락 날짜는 download 내부에서 직접 `_parse_okpos_card_workbook` → `_upsert_okpos_card_csv`로 적재(t2 재활용 안 함, 반복 재조회 방지).

```python
def download_okpos_card_test(**context) -> str:
    sale_date = _resolve_sale_date(**context)   # primary XCom(sale_date) push 포함
    download_dir = OKPOS_CARD_DOWNLOAD_DIR
    _cleanup_download_dir(download_dir)
    driver = _launch_browser(download_dir=download_dir)
    backfilled, empty_days = [], []
    try:
        wait = WebDriverWait(driver, WAIT_TIMEOUT)
        _setup_download_dir(driver, download_dir)
        _login(driver, wait)

        # primary
        downloaded = _download_one_date(driver, wait, download_dir, sale_date)
        if downloaded is None:
            raise TimeoutException(f"OKPOS card primary download 실패: sale_date={sale_date}")
        context["ti"].xcom_push(key="downloaded_path", value=str(downloaded))

        # 누락 lookback 자동 백필
        for d in _missing_lookback_dates(sale_date):
            _cleanup_download_dir(download_dir)
            f = _download_one_date(driver, wait, download_dir, d)
            if f is None:
                empty_days.append(d)
                continue
            normalized = _parse_okpos_card_workbook(f, d)
            _upsert_okpos_card_csv(normalized)
            f.unlink(missing_ok=True)
            backfilled.append(d)
            logger.info("OKPOS card 자동 백필 완료: %s | rows=%d", d, len(normalized))
    finally:
        try: driver.quit()
        except Exception: pass
    context["ti"].xcom_push(key="backfilled_dates", value=backfilled)
    context["ti"].xcom_push(key="empty_dates", value=empty_days)
    logger.info("OKPOS card download 완료: primary=%s, backfilled=%s, empty=%s", sale_date, backfilled, empty_days)
    return f"downloaded primary={sale_date}, backfilled={backfilled}, empty={empty_days}"
```
- `save_okpos_card_test_csv`(t2)는 기존대로 primary만 upsert → 그대로 둔다.
- `validate_okpos_card_lookback`(t4)는 변경 없음. 자동 백필이 구멍을 메우므로 다음 런에서 통과.

### 4. 즉시 백필 (코드 배포 후 1회)
07-03·07-04를 채우기 위해 DAG를 conf로 2회 수동 트리거하거나, 배포 후 정상 스케줄 1회 실행으로 자동 백필되게 둔다.
- 수동 트리거: conf `{"sale_date":"2026-07-03"}`, `{"sale_date":"2026-07-04"}`
- 또는 그냥 다음 정상 런 → `_missing_lookback_dates`가 07-03·07-04를 잡아 자동 수집.

## Constraints
- 로그인/브라우저 기동은 세션당 1회만 (`_download_one_date` 내부에서 `_launch_browser`/`_login` 호출 금지).
- `_wait_for_store_in_results` TimeoutException은 "승인 0건"으로 간주해 `None` 반환 → 전체 런을 죽이지 않는다. 단 **primary 날짜는 실패 시 raise**(정상 수집 보장).
- `min_date`(2026-04-17) 이전 날짜는 백필 대상 제외.
- 자동 백필은 누락 수만큼 셀레늄 조회를 늘리므로 DAG `execution_timeout`(현재 40분) 초과 주의. 보통 누락 0~1건이라 영향 미미.
- 파일/폴더 구조·파일명 변경 금지. `_upsert_okpos_card_csv`의 파티션 규칙 그대로 사용.
