# 네이버 자동 수집 대시보드 추가

## Task
`doridang_collector_개발용` 크롬 확장(MV3, Multi-Site Collector)에 **네이버 스마트플레이스 통계 수집 대시보드**를 추가한다. 기본은 어제 하루를 수집하고, 날짜를 지정하면 시작~종료 구간을 **하루씩(day-by-day)** 순차 조회해 수집한다. 진행/성공·실패/예외를 실시간으로 보여주는 **라이브 로그** 패널을 반드시 포함해 막힌 지점을 바로 복구할 수 있어야 한다.

- 프로젝트 루트: `C:\Users\민준\OneDrive - 주식회사 도리당\Extention\doridang_collector_개발용`
- 대상 페이지(CSR/SPA):
  `https://new.smartplace.naver.com/bizes/place/{placeId}/statistics?bookingBusinessId={bId}&endDate={d}&startDate={d}&term=daily`
- 매장: **송파삼전점 1개만** 하드코딩 (placeId `11835505`, bookingBusinessId `1638183`)
- 로그인: **브라우저에 네이버 이미 로그인 전제** → 대시보드는 URL 이동만. 매장별 계정 로그인 불필요.
- 저장: **CSV 다운로드** (기존 배민/쿠팡과 동일, `background.js`의 `DOWNLOAD_CSV` 경유)

## Project Conventions
- 이 프로젝트는 Airflow/Python이 아니라 **크롬 확장(순수 JS, MV3)** 이다. Python 규칙 적용 금지.
- 콘텐츠 스크립트 로드 순서 고정: `00_config.js`(전역 `Utils={}`,`Sites={}`) → `01_utils.js`(Utils 채움) → 플랫폼 모듈(`02_baemin.js`,`03_coupangeats.js`,…, `Sites['x']` 등록) → `04_auto_login.js` → `05_main.js`(메시지 핸들러/디스패치) → `06_batch.js`
- 플랫폼 모듈 규약: `Sites['name'] = { name, match(url), _stopFlag, async collect(opts), ... }`. 디스패치는 `05_main.js`가 런타임에 `site.match(url)`로 수행 → **js 배열 등록 순서는 무관**.
- 메시지 계약: 러너/팝업 `--COLLECT-->` `05_main.js` → `Sites[x].collect()`; 콘텐츠 `--DOWNLOAD_CSV-->` `background.js`; 라이브 로그는 `{type:'LIVE_LOG', msg}`.
- 저장 파일명 규약: `background.js`가 BOM(`﻿`) 부여 후 `chrome.downloads.download`.
- `Utils`는 **콘텐츠 스크립트 전용 전역** — 러너(`runner_*.js`)에서는 사용 불가. 러너는 자체 로컬 유틸을 소량 작성.
- manual vs batch 날짜 규칙: 대시보드(batch) 경로만 날짜를 강제할 수 있다. manual은 사용자의 페이지 날짜를 유지.
- ⚠️ 이 경로는 OneDrive 동기화 폴더다. `*-DESKTOP-*` 백업 파일은 **절대 수정 금지**.

## Implementation Steps

### 1. `content/07_naver.js` — 추출 모듈 (`02_baemin.js` 패턴)
`Sites['naver']`를 등록한다. **CSV 다운로드는 하지 않고** 파싱한 1일치 결과를 `NAVER_STAT_COMPLETE` 메시지로 러너에 반환한다(구간 전체를 러너가 모아 1개 CSV로 저장).

```js
Sites['naver'] = {
  name: '네이버 스마트플레이스',
  match: (url) => url.includes('new.smartplace.naver.com') && url.includes('/statistics'),
  _stopFlag: false,

  async collect(opts = {}) {
    this._stopFlag = false;
    const target = (location.href.match(/startDate=(\d{4}-\d{2}-\d{2})/) || [])[1] || '';
    try {
      // 1) 숫자 렌더 대기 + SPA stale-read 방지: 화면 날짜 span == target 될 때까지 polling
      const ok = await this._waitForDate(target, 15000);
      if (!ok) {
        chrome.runtime.sendMessage({ type:'NAVER_STAT_COMPLETE',
          payload:{ date:target, success:false, error:'렌더/날짜 타임아웃' } });
        return;
      }
      // 2) 라벨 기준 추출 (클래스는 해시 → prefix 매칭)
      const place_inflow = this._metricByLabel('플레이스 유입');
      const reservations = this._metricByLabel('예약');
      chrome.runtime.sendMessage({ type:'NAVER_STAT_COMPLETE',
        payload:{ date:target, place_inflow, reservations, success:true } });
    } catch (e) {
      chrome.runtime.sendMessage({ type:'NAVER_STAT_COMPLETE',
        payload:{ date:target, success:false, error:String(e && e.message || e) } });
    }
  },

  // 라벨 텍스트를 포함하는 카드/컨테이너를 찾아 그 안의 number를 읽음
  _metricByLabel(labelText) {
    const nums = document.querySelectorAll('[class*="ReportSummary_number"]');
    for (const el of nums) {
      // el 주변(부모 2~3단계) 텍스트에 라벨이 있으면 매칭
      let node = el;
      for (let up = 0; up < 4 && node; up++) {
        if (node.textContent && node.textContent.includes(labelText)) {
          const m = (el.textContent || '').match(/[\d,]+/);
          return m ? m[0].replace(/,/g, '') : '';
        }
        node = node.parentElement;
      }
    }
    return '';
  },

  // '26. 7. 20. 월' → '2026-07-20' 로 변환해 target 과 일치하고 number 가 존재할 때까지 polling
  async _waitForDate(target, timeout = 15000) {
    const t = Date.now();
    while (Date.now() - t < timeout) {
      if (this._stopFlag) return false;
      const hasNum = document.querySelector('[class*="ReportSummary_number"]');
      const shown = this._pageDateISO();
      if (hasNum && (!target || shown === target)) return true;
      await new Promise(r => setTimeout(r, 200));
    }
    return false;
  },

  // 페이지의 '26. 7. 20. 월' 형태 span 을 ISO(yyyy-mm-dd)로 변환. 못 찾으면 ''
  _pageDateISO() {
    const spans = document.querySelectorAll('span');
    for (const s of spans) {
      const m = (s.textContent || '').match(/(\d{2})\.\s*(\d{1,2})\.\s*(\d{1,2})\./);
      if (m) {
        const yyyy = '20' + m[1];
        return `${yyyy}-${String(m[2]).padStart(2,'0')}-${String(m[3]).padStart(2,'0')}`;
      }
    }
    return '';
  }
};
```
> 라벨↔값 대응(플레이스 유입 vs 예약)은 **1차 실행 원시 로그로 반드시 검증**. `_metricByLabel`이 못 잡으면 라벨 근접 탐색 깊이(`up<4`)나 라벨 문자열을 조정.

### 3. `runner_naver.js` — 오케스트레이션 (날짜 루프)
`runner_baemin.js`의 탭 헬퍼(`waitTabComplete`,`injectContent`,`sleepWithStop`,`log`)를 그대로 이식하되 CONTENT_FILES에 `07_naver.js` 포함. **매장 루프 대신 날짜 루프**.

```js
(() => {
  'use strict';
  const STORE = { name:'송파삼전점', placeId:'11835505', bookingBusinessId:'1638183' };
  const CONTENT_FILES = [
    'content/00_config.js', 'content/01_utils.js',
    'content/07_naver.js', 'content/05_main.js'
  ];
  const urlFor = (d) =>
    `https://new.smartplace.naver.com/bizes/place/${STORE.placeId}/statistics` +
    `?bookingBusinessId=${STORE.bookingBusinessId}&endDate=${d}&startDate=${d}&term=daily`;

  let stopRequested = false, pending = null;

  // ── 러너 로컬 유틸 (Utils는 콘텐츠 전용이라 여기선 못 씀) ──
  const sleep = (ms) => new Promise(r => setTimeout(r, ms));
  async function sleepWithStop(ms){ const t=Date.now(); while(Date.now()-t<ms){ if(stopRequested) return; await sleep(100);} }
  function ymd(dt){ return `${dt.getFullYear()}-${String(dt.getMonth()+1).padStart(2,'0')}-${String(dt.getDate()).padStart(2,'0')}`; }
  function yesterday(){ const d=new Date(); d.setDate(d.getDate()-1); return ymd(d); }
  function eachDay(startStr, endStr){
    const out=[]; let s=new Date(startStr+'T00:00:00'), e=new Date(endStr+'T00:00:00');
    if (e < s) { const t=s; s=e; e=t; }
    for (let d=new Date(s); d<=e; d.setDate(d.getDate()+1)) out.push(ymd(new Date(d)));
    return out;
  }
  function escapeCSV(v){ v=String(v==null?'':v); return /[",\n]/.test(v)?`"${v.replace(/"/g,'""')}"`:v; }
  function toCSV(headers, rows){ let c=headers.join(',')+'\n'; for(const r of rows) c+=headers.map(h=>escapeCSV(r[h])).join(',')+'\n'; return c; }
  function log(msg){ const el=document.getElementById('log'); const line=`[${new Date().toLocaleTimeString()}] ${msg}\n`; el.textContent+=line; el.scrollTop=el.scrollHeight; }

  // ── 탭 헬퍼 (runner_baemin.js 이식) ──
  async function getUrl(tabId){ try{ return (await chrome.tabs.get(tabId)).url||''; }catch(_){ return ''; } }
  async function waitTabComplete(tabId, timeoutMs=15000){
    const dl=Date.now()+timeoutMs;
    while(Date.now()<dl){ if(stopRequested) return false;
      try{ const t=await chrome.tabs.get(tabId); if(t.status==='complete'&&/^https?:/.test(t.url||'')) return true; }catch(_){}
      await sleep(500); }
    return false;
  }
  async function injectContent(tabId){
    for(let a=1;a<=5;a++){
      try{ const r=await chrome.tabs.sendMessage(tabId,{type:'PING'}); if(r?.success) return true; }catch(_){}
      await waitTabComplete(tabId,10000);
      try{ await chrome.scripting.executeScript({target:{tabId},files:CONTENT_FILES});
        await sleepWithStop(900);
        const r=await chrome.tabs.sendMessage(tabId,{type:'PING'}); if(r?.success) return true;
      }catch(e){ log(`content 주입 실패(${a}/5): ${e.message}`); }
      await sleepWithStop(1000+a*500);
    }
    return false;
  }

  // NAVER_STAT_COMPLETE / LIVE_LOG 수신
  chrome.runtime.onMessage.addListener((msg) => {
    if (msg?.type==='LIVE_LOG'){ log(msg.msg); return; }
    if (msg?.type==='NAVER_STAT_COMPLETE' && pending && !pending.settled){
      pending.settled=true; pending.resolve(msg.payload); }
  });
  function waitStat(timeoutMs=30000){
    return new Promise(resolve=>{
      pending={settled:false,resolve};
      setTimeout(()=>{ if(!pending.settled){ pending.settled=true; resolve({success:false,error:'timeout'}); } }, timeoutMs);
    });
  }

  async function run(){
    stopRequested=false;
    const days=eachDay(document.getElementById('startDate').value, document.getElementById('endDate').value);
    const rows=[]; let failed=0;
    log(`총 ${days.length}일 수집 시작 (${STORE.name})`);
    const tab=await chrome.tabs.create({ url:urlFor(days[0]), active:false });
    try{
      for(const d of days){
        if(stopRequested){ log('중단됨'); break; }
        log(`▶ ${d} 조회…`);
        await chrome.tabs.update(tab.id,{ url:urlFor(d) });
        await waitTabComplete(tab.id);
        if(!await injectContent(tab.id)){ failed++; log(`✗ ${d} content 주입 실패`); continue; }
        const w=waitStat(30000);
        try{ await chrome.tabs.sendMessage(tab.id,{ type:'COLLECT', source:'batch' }); }catch(e){ log(`COLLECT 전송 실패: ${e.message}`); }
        const res=await w;
        if(res.success){
          rows.push({ store:STORE.name, date:d, ym:d.slice(0,7).replace('-','_'),
                      place_inflow:res.place_inflow, reservations:res.reservations });
          log(`✓ ${d} 유입 ${res.place_inflow} / 예약 ${res.reservations}`);
        } else { failed++; log(`✗ ${d} 실패: ${res.error||'?'}`); }  // 실패해도 다음 날짜 계속
        await sleepWithStop(800);
      }
    } finally {
      if(rows.length){
        const csv=toCSV(['store','date','ym','place_inflow','reservations'], rows);
        const fn=`naver_stat_${STORE.name}_${days[0]}_${days[days.length-1]}.csv`;
        chrome.runtime.sendMessage({ type:'DOWNLOAD_CSV', content:csv, filename:fn });
        log(`💾 CSV 저장: ${rows.length}일 → ${fn}`);
      }
      if(failed===0) { try{ await chrome.tabs.remove(tab.id); }catch(_){} }  // 실패 남으면 탭 유지(검사용)
      log(`완료: 성공 ${rows.length} / 실패 ${failed}`);
    }
  }

  document.addEventListener('DOMContentLoaded', () => {
    const y=yesterday();
    document.getElementById('startDate').value=y;
    document.getElementById('endDate').value=y;
    document.getElementById('startBtn').addEventListener('click', run);
    document.getElementById('stopBtn').addEventListener('click', ()=>{ stopRequested=true; log('중단 요청'); });
    document.getElementById('openBtn').addEventListener('click', ()=> chrome.tabs.create({ url:urlFor(document.getElementById('startDate').value) }));
    document.getElementById('logBtn').addEventListener('click', ()=>{
      const blob=new Blob([document.getElementById('log').textContent],{type:'text/plain'});
      const a=document.createElement('a'); a.href=URL.createObjectURL(blob); a.download=`naver_log_${Date.now()}.txt`; a.click();
    });
  });
})();
```

### background.js (DOWNLOAD_CSV 핸들러 — 수정 불필요, 발신자 무관 동작)
```js
chrome.runtime.onMessage.addListener((msg, sender, sendResponse) => {
  if (msg?.type === 'DOWNLOAD_CSV') {
    const content = msg.content ?? '';
    const filename = sanitizeDownloadFilename(msg.filename, msg);
    const blob = new Blob(['﻿' + content], { type: msg.mimeType || 'text/csv;charset=utf-8' });
    const reader = new FileReader();
    reader.onloadend = () => {
      chrome.downloads.download({ url: reader.result, filename, saveAs: false }, (id) => {
        sendResponse(chrome.runtime.lastError ? { success:false } : { success:true, downloadId:id, filename });
      });
    };
    reader.readAsDataURL(blob);
    return true;
  }
});
```

## Verification Loop
```
LOOP until all PASS:
  1. Test Cases 1~5(자동) 순서대로 실행
  2. FAIL 항목 → 원인 분석 (문법/JSON/등록 누락)
  3. 코드 수정
  4. 1~5 재실행
  5. 전체 PASS 후 6~9(수동 E2E)는 사용자에게 실행 안내
종료 조건: 1~5 전체 PASS + Constraints 위반 없음
```

## Constraints
- Airflow/Python 규칙 적용 금지 — 순수 JS 크롬 확장.
- `Utils`(콘텐츠 전용 전역)를 러너에서 직접 호출하지 말 것. 러너는 로컬 유틸(`toCSV`,`eachDay`,`ymd`)을 자체 보유.
- `background.js`는 **수정하지 않는다** (기존 `DOWNLOAD_CSV` 핸들러 재사용).
- `content/05_main.js`, `runCollector` 디스패치는 **수정 불필요** (런타임 `match`로 자동 인식).
- 콘텐츠 클래스명 `ReportSummary_number__DXOlD`는 CSS-module 해시 → **정확 클래스 매칭 금지**, `[class*="ReportSummary_number"]` prefix 매칭 사용.
- CSR/SPA이므로 로드 직후 `querySelector`는 null → `_waitForDate` polling 필수. 날짜 span == target 확인으로 stale-read 방지.
- `date`는 URL `startDate`(yyyy-mm-dd)를 신뢰, `ym`은 `date.slice(0,7).replace('-','_')`.
- `*-DESKTOP-*` 백업 파일 절대 수정 금지. 변경 최소화.
- 다매장/accounts.js 확장은 이번 범위 아님 (STORE 단일 상수 유지).
