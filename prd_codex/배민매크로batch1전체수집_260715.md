## Implementation Steps

1. **schedule import 정리** (현재 63~67행)
   - `SMD_BAEMIN_COLLECT_BATCH2_TIME`, `SMD_BAEMIN_COLLECT_BATCH3_TIME`를 import에서 제거하고 `SMD_BAEMIN_COLLECT_BATCH1_TIME`만 남긴다.
   ```python

       SMD_BAEMIN_COLLECT_BATCH1_TIME,
   )
   ```

2. **헬퍼 시그니처 타입 완화** (`_build_baemin_macro_dag`, 현재 1027~1032행)
   - `collect_range: str` → `collect_range: str | None`

3. **batch1을 전체 수집으로 변경** (현재 1108~1112행)
   - `collect_range="batch:1/3"` → `collect_range=None`
   - `_split_accounts_by_range`는 collect_range가 None/빈값이면 분할 없이 전체 반환하므로 13개 전부 수집된다.
   ```python
   dag = _build_baemin_macro_dag(
       dag_id_value=dag_id,
       schedule=SMD_BAEMIN_COLLECT_BATCH1_TIME,
       collect_range=None,
   )
   ```

4. **batch2/batch3 DAG 정의 삭제** (현재 1114~1124행)
   - `dag_batch2 = _build_baemin_macro_dag(...)` 블록 전체 삭제
   - `dag_batch3 = _build_baemin_macro_dag(...)` 블록 전체 삭제
   - Batch2/Batch3 dag_id는 이 파일 안에서만 참조됨(외부 의존 없음, grep 확인 완료) → 안전하게 제거

5. **Airflow 메타데이터 정리** (코드 반영 후, 컨테이너에서 실행)
   ```bash
   docker exec airflow-airflow-scheduler-1 airflow dags delete DB_Beamin_Macro_Dags_Batch2 -y
   docker exec airflow-airflow-scheduler-1 airflow dags delete DB_Beamin_Macro_Dags_Batch3 -y
   ```

```python
def load_accounts(target_stores: list[str], exact: bool = False) -> list[dict]:
    """sales_employee.csv → 배달의 민족 계정 목록. target_stores 비면 전 매장."""
    df = load_automation_account_df(platform="배달의 민족", target_stores=target_stores, exact=exact)
    accounts = [
        {"account_id": str(r["계정ID"]).strip(),
         "password":   str(r["계정PW"]).strip(),
         "store_name": str(r["매장명"]).strip()}
        for _, r in df.iterrows()
        if str(r.get("계정ID","")).strip() and str(r.get("계정PW","")).strip()
    ]
    return accounts
```
