# 배민 우가클 현재월 수집 멈춤 수정

## Task
배민 우가클(우리가게 클릭) 데이터가 현재월(예: 2026-07)에서 07-10 이후 갱신되지 않는다. 전 매장 공통 증상이며, 원인은 재시도(F5/revisit) 로직이 `previous_month`에만 걸려 있어 현재월은 빈값 한 번에 즉시 포기하고, 빈값 시 `_save_csv`를 호출하지 않아 오래된 CSV가 그대로 보존되기 때문이다. 재시도 게이트를 "월 조건" 대신 "기존 파일 존재" 기준으로 확장해 현재월도 재시도하게 만든다.

## Implementation Steps

1. **F5 재시도 게이트 확장 (553~562행)**
   현재 조건 `if not rows and (ym == previous_month and existing_count > 0):` 를
   `if not rows and existing_count > 0:` 로 변경.
   → 현재월이라도 기존 CSV에 실데이터가 있던 매장이 빈값으로 읽히면 일시적 오판으로 보고 F5 재시도.

2. **revisit 재시도 게이트 확장 (564~565행)**
   현재 조건 `if not rows and (ym == previous_month and existing_count > 0):` 를
   `if not rows and existing_count > 0:` 로 변경.
   → F5 후에도 빈값이면 revisit(홈 복귀 후 타임스탬프 재방문) 재시도.

3. **정상 빈값 매장·신규 매장은 현행 유지**
   `existing_count > 0` 조건이 이미 신규 매장(파일 없음, count=0)과
   광고 미집행 정상 빈값(첫 수집)에는 재시도가 걸리지 않게 자연스럽게 필터링함.
   따라서 별도 `_woori_empty_is_expected` 분기 추가 불필요.

4. **previous_month 전용 RuntimeError 가드(578~582행)는 그대로 둔다.**
   현재월까지 확대하면 월초 정상 빈값에 과민 실패할 수 있으므로 확대 금지.

## Reference Code
### DB_Beamin_02_woori_shop_click.py (활성 collect_woori_for_driver, 523~585행)
```python
def collect_woori_for_driver(driver, store_list: list[dict]) -> None:
    """Override earlier implementation with stronger empty-page recovery."""
    target_months = _get_target_months()  # [이번달, 저번달]
    previous_month = pendulum.now(KST).subtract(months=1).format("YYYY-MM")

    for store_info in store_list:
        if store_info.get("brand") not in KNOWN_BRANDS:
            logger.info("우가클 광고 없는 브랜드 스킵: %s (%s)", store_info["store"], store_info.get("brand"))
            continue
        store_id = store_info["store_id"]
        for ym in target_months:
            url = (f"https://self.baemin.com/shops/{store_id}"
                   f"/stat/marketing/woori-shop-click?initialDateOption=MONTHLY&initialMonth={ym}")
            logger.info("수집 이동: %s %s", store_info["store"], ym)
            _navigate_to_woori_url(driver, store_id, ym, url)
            time.sleep(_PAGE_SETTLE)
            rows = _extract_table_rows(driver, store_id, f"{store_info['brand']} {store_info['store']}", ym)
            existing_count = _existing_woori_row_count(store_info["brand"], store_info["store"], ym)

            if not rows and (ym == previous_month and existing_count > 0):   # ← 수정 대상 1
                logger.info("재시도1 (F5): %s %s", store_info["store"], ym)
                try:
                    driver.refresh()
                except Exception as e:
                    if _is_driver_crash_error(e): raise
                    logger.info("우가클 F5 실패(빈값 처리 계속) (%s %s): %s", store_info["store"], ym, e)
                time.sleep(8.0)
                rows = _extract_table_rows(driver, store_id, f"{store_info['brand']} {store_info['store']}", ym)

            if not rows and (ym == previous_month and existing_count > 0):   # ← 수정 대상 2
                rows = _retry_woori_after_empty(driver, store_info, ym, url)

            if rows:
                saved = _save_csv(rows, store_info["brand"], store_info["store"], ym)
                logger.info("저장 완료: brand=%s store=%s ym=%s -> %s", store_info["brand"], store_info["store"], ym, saved)
            else:
                if _woori_empty_is_expected(store_info):
                    logger.info("우가클 데이터없음(정상): brand=%s store=%s ym=%s", store_info["brand"], store_info["store"], ym)
                else:
                    logger.info("데이터 없음(정상): brand=%s store=%s ym=%s", store_info["brand"], store_info["store"], ym)
                if ym == previous_month and existing_count > 0 and not _woori_empty_is_expected(store_info):
                    raise RuntimeError(f"woori previous month unexpectedly empty: {store_info['store']} {ym} existing_rows={existing_count}")
            time.sleep(random.uniform(1.5, 3.0))
```

def _existing_woori_row_count(brand, store, ym) -> int:
    out_path = _woori_csv_path(brand, store, ym)
    if not out_path.exists(): return 0
    try: return len(pd.read_csv(out_path, dtype=str))
    except Exception as e:
        logger.warning("기존 우리가게 클릭 CSV 읽기 실패 (%s): %s", out_path, e); return 0

## Constraints
- `_retry_woori_after_empty`, `_extract_table_rows`, `_save_csv`, `_navigate_to_woori_url` 등 기존 함수 시그니처 변경 금지
- 578~582행 previous_month RuntimeError 가드는 **그대로 유지** (현재월로 확대 금지)
- `_FAST_EMPTY_TIMEOUT`(8s)·`_TABLE_TIMEOUT`(18s)·`_PAGE_SETTLE`(4.0) 상수는 이번 수정에서 건드리지 않음 (회귀 위험)

- 로직 외 리팩터링·주석 대량 추가 금지
