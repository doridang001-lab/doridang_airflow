## 적재 형식 (조사 확정 — 이대로 저장)
| raw 파일 (E:\down, Collect_Data) | 파티션 루트 | 저장 파일 | 형식 |
|---|---|---|---|
| `coupangeats_orders_*.csv` | `COUPANG_ORDERS_DB` (= coupang_macro/orders) | `orders_{ym}.parquet` | parquet + `_row_hash` |
| `coupangeats_cmg_*.csv` | `COUPANG_ORDERS_DETAIL_DB/"cmg"` | `cmg.csv` | csv (utf-8-sig) |
| `coupangeats_options_*.csv` | `COUPANG_ORDERS_DETAIL_DB/"options"` | `options.csv` | csv (utf-8-sig) |

- 파티션 경로: `{루트}/brand={brand}/store={store}/ym={ym}/{파일}`
- **brand/store 도출**: raw 표시명(`닭도리탕 전문 도리당 광명철산점`, `나홀로 1인 곱도리탕 강동점`)에서
  - `brand` = 표시명에 포함된 브랜드 키워드(`도리당` 또는 `나홀로`)
  - `store` = 마지막 공백 토큰(`광명철산점`, `강동점`). options는 매장명 끝 `(946241)` 괄호 storeid 제거 후 마지막 토큰.
  - 도출 후 `store_normalize.lookup_store_key(brand, store)` 로 정규화(예: `나홀로 대신점`→`부산대신점`). 정규화 결과를 최종 store로 사용.
- **ym 도출**: orders=`order_date`(`2026.06.16 21:48`→`2026-06`), cmg=`조회일자`(`2026-06-10`→`2026-06`), options=날짜컬럼 없음 → **파일명 끝 날짜 토큰(`20260617`)의 `YYYY-MM`** 사용.
- raw 컬럼(헤더는 utf-8-sig BOM 포함):
  - orders: `collected_at,store_id,store_name,order_date,order_id,delivery_type,order_status,order_summary,total_price,is_cancelled,menu_name,menu_qty,menu_price,menu_options,매출액,상점부담_쿠폰,중개_이용료,결제대행사_수수료,배달비,광고비,부가세,즉시할인금액,정산_예정_금액,취소금액`
  - cmg: `collected_at,조회일자,매장명,광고상태,광고기간,광고비율,광고비용,신규고객,광고주문수,광고매출,전체매출,광고클릭수,광고노출수`
  - options: `매장명,옵션그룹,적용메뉴,옵션명,옵션가격,노출여부`

  KNOWN_BRANDS = ["도리당", "나홀로"]

- `_iter_source_files(prefix)`: `DOWN_DIR/f"coupangeats_{prefix}_*.csv"` + `COLLECT_SRC/f"coupangeats_{prefix}_*.csv"` 글롭. 각 파일을 `{"path": Path, "source": "down"|"collect"}` 로 반환.
- `_resolve_brand_store(display_name) -> tuple[str,str]`:
  ```python
  name = re.sub(r"\(\d+\)\s*$", "", str(display_name)).strip()  # options 괄호 storeid 제거
  brand = next((b for b in KNOWN_BRANDS if b in name), "")
  store_token = name.split()[-1] if name.split() else name
  store = lookup_store_key(brand, store_token) or store_token
  return brand, store
  ```
  미매칭(brand=="" 또는 store_token 비정상)이면 `logger.warning` 으로 표면화.
- `_ym_from_order_date(s)`: `"2026.06.16 21:48"` → `"2026-06"` (`s[:7].replace(".", "-")`).
- `_ym_from_iso(s)`: `"2026-06-10"` → `"2026-06"` (`str(s)[:7]`).
- `_ym_from_filename(path)`: 파일명에서 `(\d{8})` 마지막 매치 → `YYYYMM` → `f"{y}-{m}"`.
- `_load_orders(files) -> dict`:
  - 각 파일 `pd.read_csv(encoding="utf-8-sig", dtype=str)`.
  - `brand,store = _resolve_brand_store(row["store_name"])`, `ym = _ym_from_order_date(row["order_date"])` (행 단위 파생; 보통 파일 내 단일 매장이나 안전하게 행 기준).
  - `_row_hash` = 모든 원본 컬럼 join 후 `hashlib.md5(...).hexdigest()[:12]` (기존 파티션 포맷과 동일하게 12자리).
  - `(brand,store,ym)` groupby → 파티션 디렉터리 생성 → 기존 `orders_{ym}.parquet` 있으면 read 후 concat → `_row_hash` dedup(keep="last") → `to_parquet(index=False)`.
- `_load_csv_dataset(files, out_dir, out_name, ym_fn, dedup_subset) -> dict`: cmg/options 공용.
  - read_csv(utf-8-sig) → `brand,store = _resolve_brand_store(row["매장명"])`, `ym = ym_fn(...)`.
  - 파티션 디렉터리 `{out_dir}/brand=/store=/ym=/` 생성 → 기존 `{out_name}.csv` concat → `drop_duplicates(subset=dedup_subset, keep="last")` → `to_csv(index=False, encoding="utf-8-sig")`.
  - cmg dedup_subset=`["매장명","조회일자"]`, options dedup_subset=`["매장명","옵션그룹","옵션명","적용메뉴"]`.
- `_cleanup_sources(loaded_files)`: 적재 성공한 파일만.
  - `source=="down"` → `Path.unlink()` (삭제).
  - `source=="collect"` → `ARCHIVE_DIR.mkdir(parents=True, exist_ok=True)` 후 `shutil.move(...)` (보관 이동, 삭제 금지).
- `load_coupang_macro_partition()`:
  - orders/cmg/options 각각 `_iter_source_files` → load → 성공 파일 리스트 수집 → `_cleanup_sources` → `json.dumps({...요약...}, ensure_ascii=False)` 반환.
  - 빈 소스면 "처리할 파일 없음" 요약 반환(에러 아님).

### modules/transform/utility/store_normalize.py
```python
STORE_NAME_MAP: dict[str, str] = {
    "도리당 일산백석점": "도리당 백석점",
    "나홀로 대신점": "나홀로 부산대신점",
    "도리당 간석중앙점": "도리당 인천간석중앙점",
    # ... (구버전→현재 이름 매핑, "도리당 " 접두어 포함)
}
def normalize(series: pd.Series) -> pd.Series:
    """매장명 Series를 현재 공식명으로 치환. 접두어 유무 무관."""
    return series.replace(_MAP)
def lookup_store_key(brand: str, store: str) -> str:
    """입력 브랜드/지점명을 매핑 테이블 기반으로 정규화. 접두어 제거한 지점 키 반환."""
    normalized_store = f"{brand} {store}".strip()
    if not normalized_store:
        return ""
    normalized = _MAP.get(normalized_store, normalized_store)
    prefix = f"{brand} ".strip()
    if brand and normalized.startswith(prefix):
        return normalized[len(prefix):].strip()
    return store.strip()
```

### modules/transform/pipelines/db/DB_Coupang_01_orders.py (파티션 저장 패턴 참고)
```python
from modules.transform.utility.paths import COUPANG_ORDERS_DB
def save_orders_csv(rows: list[dict], brand: str, store: str, target_date: str) -> Path:
    ym = target_date[:7]
    out_dir = COUPANG_ORDERS_DB / f"brand={brand}" / f"store={store}" / f"ym={ym}"
    out_dir.mkdir(parents=True, exist_ok=True)
    out_path = out_dir / f"orders_{ym}.csv"
    new_df = pd.DataFrame(rows, columns=_ORDER_COLUMNS)
    new_order_ids = set(new_df["order_id"].dropna().unique())
    if out_path.exists():
        existing = pd.read_csv(out_path, dtype=str)
        existing = existing[~existing["order_id"].isin(new_order_ids)]
        combined = pd.concat([existing, new_df.astype(str)], ignore_index=True)
    else:
        combined = new_df.astype(str)
    combined.to_csv(out_path, index=False, encoding="utf-8-sig")
    return out_path
```
