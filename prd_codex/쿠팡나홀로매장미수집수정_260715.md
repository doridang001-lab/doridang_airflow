### 1. `_filterTargetStores`에 형제 나홀로 자동 포함 (핵심)
현재 구현(515-525)은 `opts.targetStores`(배치 시 = 도리당 지점명들)에 매칭되는 드롭다운 매장만 남긴다. 여기에 "타깃 매장과 **지점명(branch)** 이 같은 도리당/나홀로 매장은 포함" 규칙을 더한다.

- 지점명 추출 헬퍼 신설 `_storeBranchKey(name)`:
  - `name` trim 후 마지막 공백 토큰을 지점명으로 사용. `"도리당 경북상주점"`→`"경북상주점"`, `"나홀로 1인 곱도리탕 경북상주점"`→`"경북상주점"`, `"나홀로1인 곱도리탕 전주전북대점"`→`"전주전북대점"`.
  - 예: `const parts = String(name||'').replace(/\s+/g,' ').trim().split(' '); return parts.length ? parts[parts.length-1] : '';`
- `_filterTargetStores` 수정 로직:
  - `targetSet`(정규화된 타깃 집합)이 없으면 기존대로 `_isTargetBrandStoreName` 전체 반환(전체 매장 모드 — 이미 나홀로 포함, 변화 없음).
  - `targetSet`이 있으면, 타깃들의 지점명 집합 `targetBranches`를 만든다.
  - 후보 매장 S를 다음 중 하나면 포함:
    - (기존) 정규화 매칭: `normalized === target || normalized.includes(target) || target.includes(normalized)`
    - (신규) `_isTargetBrandStoreName(S)` 이면서 `targetBranches.has(_storeBranchKey(S))`
  - 타 브랜드(뒤뜰애/육회상쾌통쾌/붉은참치/진리국밥/김치볶음밥 등 덕진점)는 `_isTargetBrandStoreName`에서 이미 제외 → 오포함 없음.
- 이 한 곳 수정으로 orders(`_collectOrdersMultiStore`)·CMG(`_collectCMGMultiStore`) 두 경로가 모두 나홀로를 대상에 포함(둘 다 `_filterTargetStores` 호출).

## Reference Code
### content/03_coupangeats.js — 재사용 헬퍼 & 수정 대상
```javascript
_isTargetBrandStoreName(text) {
  const normalized = String(text || '').replace(/\s+/g, ' ').trim();
  return /(^|[^가-힣A-Za-z0-9])(도리당|나홀로)/.test(normalized);
},

_normalizeStoreName(text) {
  let normalized = String(text || '')
    .replace(/\s+/g, '').replace(/[·ㆍ.,]/g, '').trim();
  normalized = normalized.replace(/^닭도리탕전문/, '');
  normalized = normalized.replace(/구로디지털단지점/g, '구로디지털점');
  return normalized;
},

_targetStoreSet(opts = {}) {
  if (!Array.isArray(opts.targetStores) || opts.targetStores.length === 0) return null;
  return new Set(opts.targetStores.map(s => this._normalizeStoreName(s)).filter(Boolean));
},

// ▼ 수정 대상: 형제 나홀로 자동 포함 규칙 추가
_filterTargetStores(storeNames, opts = {}) {
  const targetSet = this._targetStoreSet(opts);
  const brandStores = (storeNames || []).filter(storeName => this._isTargetBrandStoreName(storeName));
  if (!targetSet) return brandStores;
  const candidates = storeNames || [];
  const targets = [...targetSet];
  return candidates.filter(storeName => {
    const normalized = this._normalizeStoreName(storeName);
    return targets.some(target => normalized === target || normalized.includes(target) || target.includes(normalized));
  });
},
```
