### 1. `_transform_to_unified` 함수 수정 (line ~220)

```python
# 기존 (제거)

# 변경 후
cancel_mask = out["sale_type"] == "취소"
```

### 2. `_recalculate_order_fields` 함수 수정

order_cnt 대표행 설정에서 취소행 −1 → 0으로 변경:

```python
# 기존 (제거)
df.loc[representative_indexes, "order_cnt"] = df.loc[representative_indexes, "sale_type"].map(
    lambda value: -1 if str(value) == "취소" else 1
)

# 변경 후
df.loc[representative_indexes, "order_cnt"] = df.loc[representative_indexes, "sale_type"].map(
    lambda value: 0 if str(value) == "취소" else 1
)
```
