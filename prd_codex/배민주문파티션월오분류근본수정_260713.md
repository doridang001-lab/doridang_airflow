# 배민 주문 파티션 월 오분류 근본 수정 + 송파삼전점 원본 교체

## Task
배민 자동수집 파이프라인이 주문을 **수집일(target_date) 월** 파티션에 저장(`ym = target_date[:7]`)해서,
주문시각이 지난달인 주문이 이번달에 수집되면 엉뚱한 월 파일에 적재된다. 그 결과 월 오분류·크로스먼스
중복·특정 월 누락이 발생하고 다운스트림(validate/unified) 로직이 오작동한다. 파티션 월을 **주문시각 월**
기준으로 통일(근본 수정)하고, 기존 위반분을 재파티션으로 정리하며, 송파삼전점은 수동 재수집본(정답)으로 교체한다.

### 1. `beamin_store_io.py` — 공용 헬퍼 추가
```python
def order_ym(series: pd.Series) -> pd.Series:
    """'2026. 06. 30. (화) ...' -> '2026-06' (파싱 실패는 '')."""
    m = series.astype(str).str.extract(r"^\s*(\d{4})\.\s*(\d{1,2})\.")
    return (m[0] + "-" + m[1].str.zfill(2)).fillna("")
```

```python
from modules.transform.pipelines.db.beamin_store_io import read_table, write_table, order_ym

def _save_orders_csv(rows: list[dict], brand: str, store: str, target_date: str) -> Path:
    """주문시각 월별 파티션으로 upsert 저장 (같은 주문번호 기존 행은 해당 월에서 교체)."""
    new_df = pd.DataFrame(rows, columns=_COLUMNS).astype(str)
    ym_series = order_ym(new_df["주문시각"])
    ym_series = ym_series.where(ym_series.ne(""), target_date[:7])  # 파싱 실패만 폴백

    saved: list[Path] = []
    for ym, group in new_df.groupby(ym_series):
        stem = BAEMIN_ORDERS_DB / f"brand={brand}" / f"store={store}" / f"ym={ym}" / f"orders_{ym}"
        new_ids = set(group["주문번호"].unique())
        existing = read_table(stem)
        combined = (
            pd.concat([existing[~existing["주문번호"].isin(new_ids)], group], ignore_index=True)
            if existing is not None else group
        )
        out_path = write_table(combined, stem)
        logger.info("저장 완료: %s (%d행)", out_path, len(combined))
        saved.append(out_path)

    if not saved:
        return (BAEMIN_ORDERS_DB / f"brand={brand}" / f"store={store}"
                / f"ym={target_date[:7]}" / f"orders_{target_date[:7]}").with_suffix(".parquet")
    return next((p for p in saved if f"ym={target_date[:7]}" in str(p)), saved[0])
```

### modules/transform/pipelines/db/beamin_store_io.py
```python
import pandas as pd
from pathlib import Path
PARQUET_COMPRESSION = "zstd"

def _as_str_df(df: pd.DataFrame) -> pd.DataFrame:
    return df.fillna("").astype(str)

def read_table(stem_path: Path, columns=None) -> pd.DataFrame | None:
    pq = stem_path.with_suffix(".parquet"); csv = stem_path.with_suffix(".csv")
    if pq.exists(): return _as_str_df(pd.read_parquet(pq, columns=columns))
    if csv.exists(): return _as_str_df(pd.read_csv(csv, dtype=str, encoding="utf-8-sig", usecols=columns))
    return None

def read_file(path: Path, columns=None) -> pd.DataFrame:
    if path.suffix == ".parquet": return _as_str_df(pd.read_parquet(path, columns=columns))
    return _as_str_df(pd.read_csv(path, dtype=str, encoding="utf-8-sig", usecols=columns))

def write_table(df: pd.DataFrame, stem_path: Path) -> Path:
    stem_path.parent.mkdir(parents=True, exist_ok=True)
    pq = stem_path.with_suffix(".parquet")
    _as_str_df(df).to_parquet(pq, engine="pyarrow", compression=PARQUET_COMPRESSION, index=False)
    csv = stem_path.with_suffix(".csv")
    if csv.exists():
        try: csv.unlink()
        except OSError: pass
    return pq

def find_tables(base: Path, rel_glob_stem: str) -> list[Path]:
    found: dict[Path, Path] = {}
    for suffix in (".csv", ".parquet"):
        for p in base.glob(rel_glob_stem + suffix):
            found[p.with_suffix("")] = p
    return sorted(found.values())
```

### modules/transform/pipelines/db/DB_BaeminManual_load.py (정상 패턴 + 재사용 대상 _upsert_manual_orders)
```python
_ORDER_KEY = "주문번호"
_ORDER_LINE_KEY = ["주문번호","주문시각","주문내역","주문옵션상세","주문수량","주문옵션금액","결제금액"]

def _ym_from_order_time(value: str) -> str:  # <-- order_ym 으로 교체 대상
    m = re.match(r"(\d{4})\.\s*(\d{1,2})\.", str(value).strip())
    return f"{m.group(1)}-{m.group(2).zfill(2)}" if m else ""

def _upsert_manual_orders(existing, new_df):
    """주문번호별 최신 collected_at 인스턴스만 유지 후 라인키 dedup (재파티션 dedup 재사용)."""
    combined = new_df.copy() if existing is None or existing.empty else pd.concat([existing, new_df], ignore_index=True)
    combined = combined.fillna("").astype(str)
    if "collected_at" not in combined.columns: combined["collected_at"] = ""
    order_latest = combined[[_ORDER_KEY,"collected_at"]].groupby(_ORDER_KEY, dropna=False)["collected_at"].max().to_dict()
    ids = combined[_ORDER_KEY].fillna("").astype(str).str.strip()
    latest = ids.map(lambda oid: order_latest.get(oid, ""))
    keep = ids.eq("") | combined["collected_at"].fillna("").astype(str).eq(latest)
    out = combined[keep].copy()
    dedup_cols = [c for c in _ORDER_LINE_KEY if c in out.columns]
    out = out.drop_duplicates(subset=dedup_cols or None, keep="last")
    return out.reset_index(drop=True)

# 정상 파티션 저장 패턴 (_load_one_file 내부)
for ym, group in target_df.groupby(target_df["주문시각"].apply(_ym_from_order_time)):
    stem = BAEMIN_ORDERS_DB / f"brand={brand}" / f"store={store_key}" / f"ym={ym}" / f"orders_{ym}"
    combined = _upsert_manual_orders(read_table(stem), new_df)
    write_table(combined, stem)
```

## Constraints
- 파티션 stem 규격·컬럼 순서(`_COLUMNS`)·문자열 dtype 유지. parquet(zstd) 저장은 반드시 `write_table` 사용.
- 재파티션 dedup은 **라인아이템 보존**: `주문번호` naive dedup 금지, 최신 collected_at 인스턴스 유지 방식만 사용.
- 모든 parquet 변경 전 `LOCAL_DB/temp` 선백업(롤백 대비). OneDrive 원본 폴더에는 `.bak`을 만들지 않고 문제 시 로컬 백업에서 복원.
- 송파삼전점은 재파티션(A) 대상에서 제외하고 수동 CSV 교체(B)로만 처리.
- `_save_orders_csv` 반환 타입 `Path` 유지(호출부 `collect_orders_for_driver`/`collect_orders_for_account`가 로그용으로 사용).
- 정상 단일월 수집 동작이 기존과 바뀌면 안 됨(그룹 1개 → 동일 결과).
