### 변경 1 — account.py 우선으로 조회 순서 반전 (핵심 수정)
파일: `modules/transform/pipelines/sales/Sales_ToOrder_Review_collect.py` (현재 L74-91)

`_resolve_toorder_password()`에서 `get_pw("toorder", account_id)`(account.py)를 **가장 먼저** 조회한다. account.py에 값이 있으면 즉시 반환(스테일 env/Variable 마스킹 방지). account.py에 없는 계정만 env/Airflow Variable로 폴백한다.

**현재:**
```python
def _resolve_toorder_password(account_id: str) -> str:
    specific_key = f"TOORDER_PW_{str(account_id).upper()}"
    keys = (
        specific_key,
        "TOORDER_VOC_ACCOUNT_PW",
        "TOORDER_VOC_PW",
        "TOORDER_DELIVERY_ACCOUNT_PW",
        "TOORDER_DELIVERY_PW",
        "TOORDER_PW",
    )
    for key in keys:
        value = (os.getenv(key) or _get_airflow_variable(key)).strip()
        if value:
            return value
    account_pw = get_pw("toorder", account_id)
    if account_pw:
        return account_pw
    return ""
```

**→ 교체:**
```python
def _resolve_toorder_password(account_id: str) -> str:
    # account.py에 있는 계정은 account.py가 단일 소스(스테일 env/Variable 마스킹 방지) — .env.example 정책
    account_pw = get_pw("toorder", account_id)
    if account_pw:
        return account_pw
    # account.py에 없는 계정만 env/Airflow Variable 폴백
    specific_key = f"TOORDER_PW_{str(account_id).upper()}"
    keys = (
        specific_key,
        "TOORDER_VOC_ACCOUNT_PW",
        "TOORDER_VOC_PW",
        "TOORDER_DELIVERY_ACCOUNT_PW",
        "TOORDER_DELIVERY_PW",
        "TOORDER_PW",
    )
    for key in keys:
        value = (os.getenv(key) or _get_airflow_variable(key)).strip()
        if value:
            return value
    return ""
```
