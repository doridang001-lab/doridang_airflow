# 송파삼전점 홀·배달 매출 데이터 감시 리포트 (Telegram 발송)

## Task
송파삼전점의 전일 매출을 자동 집계해 **필수 6지표(총매출·주문수·객단가·신규·재구매·재구매비율)** 를 매일 09:30 텔레그램으로 발송한다. 채널(홀/배달)·시간대·메뉴로 원인을 쪼개고, 전일 대비 / 전주 동일요일 / 최근 4주 평균 비교와 복기 문장까지 하나의 리포트로 정리한다. 대상 매장은 **송파삼전점 단독**(신규/재구매 카드데이터가 이 매장만 존재).

- `MART_DB/okpos_cutomer/okpos_customer_daily.csv` — 신규/재구매 (송파삼전점 카드 기준)

3. **`_load_store_day(date_str) -> pd.DataFrame`**
   - `_unified_daily_path(date_str)` 로 parquet 경로 획득 → 없으면 빈 DF 반환
   - 로드 후 `store == STORE_NAME` & `sale_type != "취소"` 필터
   - `total_price`, `order_cnt` 는 `pd.to_numeric(...).fillna(0).astype(int)`
   - `channel` 컬럼 부여: `platform == "홀"` (또는 order_type in HALL_ORDER_TYPES) 이면 `"홀"`, 아니면 `"배달"`

4. **`_agg(df) -> dict`** — 채널 무관 집계 헬퍼
   - `매출 = int(df["total_price"].sum())`
   - `주문수 = int(df["order_id"].nunique())`  ※ hall_viz와 동일한 `nunique(order_id)` 방식. 구현 후 Test Case 2로 order_cnt 방식과 값 대조해 확정.
   - `객단가 = 매출 // 주문수 if 주문수 else 0`
   - 반환 `{"매출":..., "주문수":..., "객단가":...}`

5. **`_channel_agg(df) -> dict`** → `{"홀": _agg(홀), "배달": _agg(배달), "전체": _agg(df)}`

6. **`_customer_counts(date_str) -> dict`**
   - CSV 없으면 `{"신규":0,"재구매":0,"재구매비율":0.0}`
   - `pd.read_csv(OKPOS_CUSTOMER_CSV, dtype=str, encoding="utf-8-sig")` → `일별 == date_str` 필터 → `신규재구매 구분` value_counts
   - `재구매비율 = 재구매 / (신규+재구매)` (분모 0 방어), 백분율 소수1자리

7. **`_time_slot_agg(hall_df) -> dict`**
   - `classify_hall_time_slots(hall_df)` 호출 → `time_slot` 컬럼 생성
   - 점심/저녁 각각 `_agg` → `{"점심":{...}, "저녁":{...}}`

8. **`_menu_top(hall_df, n=3) -> list`**
   - 홀 메뉴별 `item_name` 그룹 → `qty` 합, `total_price` 합
   - `total_price` 내림차순 top n, 각 항목 `전체대비비율 = 판매금액 / 홀총매출`
   - 반환 `[{"메뉴":..,"판매량":..,"판매금액":..,"비율":..}, ...]`

9. **`_pct(cur, base) -> float`** — 증감률 `(cur-base)/base*100`, base 0이면 0.0

10. **`build_report(ref_date) -> str`**
    - `ref = datetime.strptime(ref_date, "%Y-%m-%d").date()`
    - 비교일자: `prev_day = ref-1`, `prev_week = ref-7`, `avg4w_dates = [ref-7, ref-14, ref-21, ref-28]`
    - 각 일자 `_load_store_day` → `_channel_agg`. 4주 평균은 존재하는 파일들의 전체 매출/주문 평균.
    - `_customer_counts(ref_date)`, 홀 df로 `_time_slot_agg`, `_menu_top`
    - 아래 텍스트 포맷을 `"\n".join(sections)` 로 조립. **4096자 이내**. 숫자는 `f"{n:,}"` 천단위.

    ```
    [직영점 홀·배달 데이터 감시] 송파삼전점
    기준일: 2026-07-06(월)

    ■ 전일 실적(합산)  총매출 x원 / 주문 x건 / 객단가 x원
       신규 x건 · 재구매 x건 · 재구매비율 x%
    ■ 채널   홀  매출 x / 주문 x / 객단가 x
             배달 매출 x / 주문 x / 객단가 x
    ■ 전일 대비(그제)   매출 ±x% · 주문 ±x건 · 객단가 ±x%
    ■ 전주 동일요일 대비 매출 ±x% · 주문 ±x건 · 객단가 ±x%
       (최근4주 평균 대비  매출 ±x%)
    ■ 시간대(홀)  점심 매출/주문/객단가 · 저녁 매출/주문/객단가
    ■ 메뉴 TOP3(홀)  A x개 x원(x%) / B ... / C ...
    ■ 복기  <주문수 vs 객단가 원인 자동 문장 + 확인사항>
    ```
    - 복기 문장 로직(예): 전주 대비 매출 감소 시, 주문수 감소율 > 객단가 변화면 "주문수 감소 영향", 반대면 "객단가 하락 영향". 시간대 중 주문수 감소 큰 슬롯 지목.

11. **`generate_report(**context) -> str`**
    - `conf = (context.get("dag_run").conf if context.get("dag_run") else {}) or {}`
    - `ref_date = conf.get("sale_date")` 없으면 전일: `(datetime.now(ZoneInfo("Asia/Seoul")).date() - timedelta(days=1)).strftime("%Y-%m-%d")`
    - `msg = build_report(ref_date)`; `context["ti"].xcom_push(key="report", value=msg)`; `context["ti"].xcom_push(key="ref_date", value=ref_date)`
    - return 요약 문자열

12. **`send_report(**context) -> str`**
    - `ti = context["ti"]`; `msg = ti.xcom_pull(key="report")`; `ref_date = ti.xcom_pull(key="ref_date")`
    - `send_telegram(msg)`
    - `LOG_DIR.mkdir(parents=True, exist_ok=True)`; `(LOG_DIR / f"hall_monitor_{ref_date}.md").write_text(msg, encoding="utf-8")`
    - return 요약 문자열

13. **`DB_Hall_Sales_Monitor_Dags.py`** (morning_briefing 패턴)
    - `dag_id = Path(__file__).stem`, `catchup=False`, `max_active_runs=1`
    - `schedule = DB_HALL_MONITOR_TIME`, `start_date` pendulum KST, tags
    - `default_args={"on_failure_callback": on_failure_callback}` (notifier에서 import)
    - 로직 import는 `importlib.import_module("modules.transform.pipelines.db.DB_Hall_Sales_Monitor")`
    - 태스크: `generate_report >> send_report` (PythonOperator, `execution_timeout` 지정, `provide_context`/`**context`)

```python
def classify_hall_time_slots(df: pd.DataFrame) -> pd.DataFrame:
    """점심(06~16)/저녁(16~24) time_slot 컬럼 부여. 취소행(order_time 00:00) 슬롯 보정 포함."""
    df = df.copy()
    raw_hour = (df["order_time"].astype(str).str.strip()
                .str.extract(r"^(\d{2}):", expand=False)
                .pipe(pd.to_numeric, errors="coerce").astype("Int64"))
    raw_hour = raw_hour.where(raw_hour != 0, other=pd.NA)
    lunch_mask = ((raw_hour >= 6) & (raw_hour < 16)).fillna(False)
    dinner_mask = ((raw_hour >= 16) & (raw_hour < 24)).fillna(False)
    df["time_slot"] = pd.NA
    df.loc[lunch_mask, "time_slot"] = "점심"
    df.loc[dinner_mask, "time_slot"] = "저녁"
    # ... order_id 매칭 보정 후 잔여는 매출비중 큰 슬롯으로 배정, 최종 fillna("저녁")
    return df
```
