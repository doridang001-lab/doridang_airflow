# 쿠팡 취소주문 매출 처리 교정 (unified 음수 → 매출액 0)

## Task
쿠팡 macro → unified_sales 교정 파이프라인이 전체취소 주문을 `total_price`의 음수(−원금액)로 이중 차감해, 송파삼전점 2026-07-10 쿠팡이츠 매출이 259,600(정답)이 아닌 229,700으로 집계된다. 원천 `매출액` 컬럼은 이미 "정상=원금액, 전체취소=0"을 정확히 담고 있으므로(218개 parquet 전수 검증: 정상 39,898건 매출액==total_price, 전체취소 788건 매출액==0), unified의 금액 산출을 `매출액` 기준으로 바꿔 취소=0 규칙에 맞춘다.

## Implementation Steps
1. `_transform_to_unified` (약 279-296행)의 `amount_by_order` 산출을 `total_price` → `매출액` 으로 교체한다.
   - `매출액`은 주문의 헤더행에만 값이 있고 나머지 메뉴행은 빈값 → `groupby(order_id).max().fillna(0).astype(int)` 로 주문당 1값 취득.
   - 숫자 변환: `pd.to_numeric(df["매출액"].astype(str).str.replace(",", "", regex=False).str.strip(), errors="coerce")`
2. 취소 음수 처리(`if sale_type == "취소": amount = -abs(amount)`)를 **제거**한다. 전체취소는 매출액이 이미 0이므로 자연히 0이 된다.
3. `order_cnt`는 현행 유지: 취소=0, 정상=1.
4. 안전 폴백: 원천에 `매출액` 컬럼이 없거나(레거시) 특정 주문 매출액이 NaN이면 `total_price` first값으로 폴백하되, **취소면 0**(음수 아님)으로 둔다.
5. 정상주문은 매출액==total_price라 값 불변 — 회귀 없어야 함.

    # ▼ 교체 대상: total_price + 취소 음수  →  매출액(취소=0)
    amount_by_order = (
        pd.to_numeric(df["total_price"], errors="coerce")
        .fillna(0)
        .groupby(out["order_id"])
        .first()
        .astype(int)
        .to_dict()
    )
    out["order_cnt"] = 0
    for order_id, group in out.groupby("order_id", sort=False):
        if group.empty:
            continue
        idx = group.index[0]
        amount = int(amount_by_order.get(order_id, 0))
        if out.at[idx, "sale_type"] == "취소":
            amount = -abs(amount)          # ← 제거 (매출액은 이미 0)
        out.at[idx, "total_price"] = amount
        out.at[idx, "order_cnt"] = 0 if out.at[idx, "sale_type"] == "취소" else 1
    out["_pk"] = _make_unified_pk(out)

    has_maechul = "매출액" in df.columns

        .groupby(out["order_id"]).first().astype(int).to_dict()
    )
    out["order_cnt"] = 0
    for order_id, group in out.groupby("order_id", sort=False):
        if group.empty:
            continue
        idx = group.index[0]
        is_cancel = out.at[idx, "sale_type"] == "취소"
        amt = amount_by_order.get(order_id)
        if amt is None or pd.isna(amt):
            amt = 0 if is_cancel else fallback_tp.get(order_id, 0)  # 폴백: 취소=0
        out.at[idx, "total_price"] = int(amt)
        out.at[idx, "order_cnt"] = 0 if is_cancel else 1
